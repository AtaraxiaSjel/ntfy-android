# ntfy Android App
This is the Android app for [ntfy](https://github.com/binwiederhier/ntfy) ([ntfy.sh](https://ntfy.sh)). It is available 
in the [Play Store](https://play.google.com/store/apps/details?id=io.heckel.ntfy).
